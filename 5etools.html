使用 Foundry?

不？ 好吧，你應該試試！ Foundry 是現代的，比 Roll20 更好的替代品，更注重 Mod 的支持。它是 5etools 的首選的整合平臺。因此，betterR20 將不再接受除了基礎維護以外的任何更新。取而代之，開發支持已完全投入到爲 Foundry 創建一套擴展工具，即爲 Plutonium 模組。

安裝此模組，複製清單文件地址，並在 Foundry 的 MOD 安裝對話框裏粘貼。查看 Plutonium 功能頁面，以瞭解該 Mod 提供的功能的概述。

需注意的是，本站中文內容目前尚未支持在 Plutonium 模組於 Foundry VTT 中的使用。如需此功能，可以考慮加入 FVTT-CN/5etools 漢化組來貢獻工作，儘早實現此功能。

使用 Roll20?

請注意： 5etools 的 Roll20 整合， betteR20，已經不再活躍開發。取而代之，開發支持已經完全投入到 Plutonium，如上文所述。

安裝 Roll20 增強套件 以及 Tampermonkey (Chrome/Edge Chromium/Firefox)，並且點擊鏈接獲取 betteR20 腳本

作爲玩家，你可以導入角色選項、法術以及物品來快速填充的角色卡。作爲 GM，你可以額外導入怪物、表格以及冒險。

爲了能夠導入怪物，你的遊戲必須擁有默認角色卡。我們強烈推薦在導入時使用 OGL，然後如果需要的話在之後切換到其他角色卡。

查看百科上的安裝指南，以及可以查看幫助頁來瞭解使用該腳本的一些基礎用法。

FAQ

5etools 有百科嗎？ 這裏（英文）。 歡迎貢獻內容。

5etools 免費嗎？ 是的！此外，你還可以在 get.5e.tools 下載源代碼。查看 自述文件 README 或者百科以獲取幫助。

我可以爲 5etools 英文站點貢獻內容嗎？ 源代碼的拷貝在 GitHub 上公開。注意，這是一個鏡像，所以並不保證內容始終最新。

我可以爲 5etools 中文站點（正在查看的此站點）貢獻內容嗎？ 源代碼的拷貝在 GitHub 上公開，可以直接爲其提供 PR。或者，你可以通過果園的 5etools 翻譯招募貼來加入 FVTT-CN 的 5etools 翻譯組。

瀏覽器與設備支持？ 最新的桌面版 Chrome/Edge Chromium/Firefox。

怎麼管理我自己的數據？ 並沒有賬號系統一說。一切都儲存在 cookies 當中。如果你清除這些 cookies，或者關閉你的隱私模式窗口，儲存的內容就會永久消失。爲了安全保存你的數據，你可以通過設置菜單下載你自己的數據。

還有別的我該知道的東西嗎？ 在鼠標懸浮時，按住 SHIFT 以就地固定住一個窗口。所有內容都有工具提示，而你作爲 DM 將永恆。
